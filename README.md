# Life Expectancy Dashboard

## Project Overview

This project presents an interactive Life Expectancy Dashboard designed to analyze global health, demographic, and socio-economic indicators across countries and years. The dashboard provides a clear view of how factors such as population size, alcohol consumption, education, income composition, disease trends, and mortality rates relate to life expectancy outcomes worldwide.

The analysis is built to support data-driven insights for public health research, policy evaluation, and academic assessment.
